# 🛒 NutriCart – Food & Diet E-Commerce Website

NutriCart is a front-end based food e-commerce website built using HTML, CSS, and JavaScript. It allows users to explore and order healthy food items and get personalized diet plans based on age and fitness goals like weight loss or weight gain.

---

## 🚀 Live Preview

👉 [Click here to visit NutriCart](https://nishakumari162.github.io/-NutriCart-Food-Diet-E-Commerce-Website/)

---

## ✨ Features

- 🧑‍🍳 Dashboard with multiple food items
- 📦 Add to Cart & View My Orders
- 📃 Product Detail View
- 🍽 Personalized Diet Plan (by age & goal)
- 🔐 Login & Registration
- 💰 Payment (Cash on Delivery)
- ✅ Order Success & Logout Pages
- 🎨 Eye-catching UI with healthy food background images

---

## 💻 Tech Stack

- *HTML5*
- *CSS3*
- *JavaScript (Vanilla)

---

## 🧾 Usage

1. Clone or download the repository
2. Open index.html in a browser
3. Navigate through dashboard, diet planner, cart, and checkout
4. Works without backend — pure front-end simulation
